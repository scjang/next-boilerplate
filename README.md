# Web App scaffolding

This scaffolding was made in TypeScript, Next js, Express js with React.

## How to use?

```bash
node -v
v14.15.0

yarn install
yarn dev
```

## Folder Structure

```
node_modules
src
├── @types
├── components
├── pages
├── server
│   └── logger
├── services
│   └── api
├── theme
├── hooks
│   └── typeName
│     └── api
│     └── updater
└── utils
```
