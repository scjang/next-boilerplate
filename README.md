# Web App scaffolding

This scaffolding was made in TypeScript, Next js, Express js with React.

## How to use?

```bash
node -v
v14.15.0

yarn install
yarn dev
```

## Folder Structure

```
node_modules
src
├── components
├── interfaces
├── pages
├── server
├── services
│   └── api
├── theme
└── utils
```
